## System of Equations 
$$\begin{align*}
g'(E) &= \sum_{k=1}^E g\left(\left\lfloor E/k \right\rfloor\right) \\\
g'(E) &= \sum_{k=1}^E \sum_{i=1}^{\lfloor E/k \rfloor} \nabla g(i) \\\
g'(E) &= \sum_{k\cdot i \le E} \nabla g(i)
\end{align*}$$
